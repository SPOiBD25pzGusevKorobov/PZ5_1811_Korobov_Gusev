# main2.py

## Описание

`main2.py` — это приложение с графическим интерфейсом на **PyQt6** для поиска калорийности продуктов питания.

Пользователь может быстро узнать калорийность продукта по его названию или коду.

Доступ к базе данных продуктов осуществляется через API.

## Основные возможности

- Поиск продуктов по названию.
- Поиск продуктов по коду.
- Отображение калорийности выбранного продукта.

## Требования

- Python 3.8+
- [PyQt6](https://pypi.org/project/PyQt6/)
- [requests](https://pypi.org/project/requests)
- Подключение к интернету для доступа к API.

## Запуск

1. Установите зависимости:
   ```
   pip install PyQt6
   # или загрузите все из requirements.txt
   pip install requests
   ```

2. Запустите интерфейс:
   ```
   python main2.py
   ```

## Что мы исправили и сделали
1. Создали интерфейс на PyQt6
2. Исправили поиск, теперь он по-нормальному ищет товар по названию
3. Для удобства встроили поиск по категориям, чтобы упростить поиск
4. Добавили поиск по тегам, для упрощения поиска

## Примечания

- Адрес API и дополнительные параметры находятся внутри файла main2.py.
- Приложение подходит для работы с русскоязычной базой продуктов питания.
